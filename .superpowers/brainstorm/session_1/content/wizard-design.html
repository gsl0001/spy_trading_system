Design: The Guided Wizard (Active Checks)

Ensuring system readiness through a pre-flight checklist

1. Deployment "Pre-flight" Checklist

Instead of just Start/Stop buttons, the Deployment page will lead with a status grid that polls the backend for these specific requirements:

IBKR Connectivity ● ONLINE

API Port 7497 detected.

ML Ensemble ● LOADED

v5.0.2 model active.

Market Data ○ STALE

Last quote > 60s ago.

Strategy Config ! REVIEW

37/37 active (Aggressive).

2. Contextual Instruction Tooltips

Every status check will have a "How to fix" hint. For example, if IBKR is offline, a tooltip will explain:

Deployment Help: IBKR Offline

  1. Open IB Gateway or TWS.
  2. Go to Settings > API > Precautions.
  3. Ensure "Bypass Order Precautions" is checked.
  4. Verify port 7497 matches config.yaml.

3. Streamlined Dashboard

The main Dashboard will be simplified to focus on "Decision Alpha" — showing exactly why the AI is or isn't trading right now, using a confidence meter.
